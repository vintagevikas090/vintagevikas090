# 🚀 Featured Projects

### 🎤 AI Interview Copilot

An AI-powered interview preparation platform that analyzes resumes against job descriptions, conducts role-specific mock interviews, evaluates candidate responses, and generates ATS scores, skill-gap analysis, and hiring recommendations.

**Tech:** LangChain • LLMs • Prompt Engineering • Structured Outputs • Streamlit

---

### 🎥 YouTube Q&A Assistant

A conversational RAG application that transforms YouTube videos into searchable knowledge bases using transcript retrieval, semantic search, chat history, and context-aware question answering.

**Tech:** LangChain • ChromaDB • HuggingFace Embeddings • Groq • Conversational RAG • Streamlit

---

### 📄 PaperLens

A local-first Research Paper Assistant that enables semantic search, document summarization, and question answering over research papers using Retrieval-Augmented Generation and on-device LLMs.

**Tech:** LangChain • FAISS • Ollama • Semantic Search • RAG • Streamlit
